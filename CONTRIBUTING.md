# Contributing

Garmon World is in an early public research-preview stage.

The best contribution right now is calm technical feedback on the public review surface.

## Useful Contributions

Helpful contributions include:

- clearer wording;
- safer public framing;
- questions for reviewers;
- suggestions for a minimal body-contour artifact;
- suggestions for a later event-reading demo artifact;
- evaluation ideas for Body Shadow / contour distinction;
- references to relevant public work.

## Before Opening An Issue

Please read:

- `README.md`
- `docs/PUBLIC_CURRENT_STATUS.md`
- `docs/PUBLIC_BODY_SHADOW_CHECKS.md`
- `docs/PUBLIC_ENGINEERING_CONTRACTS.md`
- `docs/PUBLIC_MEMORY_RESIDUE_BOUNDARIES.md`
- `docs/FOR_REVIEWERS.md`

## Good Issue Style

Good issues are:

- specific;
- calm;
- public-safe;
- focused on one point;
- written as feedback or a question, not pressure.

Examples:

> The Body Shadow framing would be easier to review if the contour comparison table used fewer terms.

> The boundary between body contour and command authority is clear here, but less clear in this paragraph: [...].

## Please Avoid

Please avoid:

- asking for private files;
- asking for raw logs;
- asking for local paths;
- asking for unpublished working notes;
- asking for exact private implementation details;
- asking for internal report names or protected hashes;
- treating static/shadow evidence as live behavior;
- framing the project as a finished product;
- pushing for strong claims before a public artifact exists.

## Pull Requests

Pull requests may be useful later.

For now, small documentation suggestions are more useful than code changes.

If you propose a change, please keep it modest and public-safe.

Good pull requests should improve clarity without exposing private mechanisms.

They should not turn static/shadow evidence into live behavior claims.

## Tone

The desired tone is:

- achievement-aware;
- careful;
- technical;
- low-pressure;
- honest about limits.

The goal is not hype. The goal is better review of a real public-safe achievement: body-contour distinction before speech.
