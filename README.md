# Garmon World

Public-safe notes for **Garmon**, an early R&D prototype exploring behavioral runtime layers for LLM-based agents.

Garmon starts from a simple separation:

> the language model speaks, but behavior should not be driven by prompt text alone.

The project explores how an agent can use explicit runtime signals, context traces, boundary checks, and auditable behavior steps before the final answer is written by an LLM.

## What This Repository Is

This repository is a public entry point for:

- a clear description of the project;
- public-safe notes;
- review questions;
- early demo planning;
- future reproducible materials.

It is not the private workbench of the system.

## What Garmon Is Not Claiming

Garmon is not presented as:

- a finished product;
- a new trained language model;
- a claim of subjective experience;
- a claim of mature autonomy;
- a claim of general intelligence;
- a replacement for external technical review.

The current public framing is modest:

> Garmon is an early research prototype for studying behavior selection around LLM agents.

## Current Public Direction

The useful question is not whether an LLM can produce impressive text.

The useful question is:

> can an agent show more reviewable behavior when speech is separated from a small runtime layer that carries context traces, boundaries, and behavior checks?

This repository will grow slowly and carefully.
