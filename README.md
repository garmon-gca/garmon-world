# Garmon World

**Where meaning becomes choice.**

Garmon is a research architecture for studying how an artificial answer can have an origin before speech.

Most LLM-based systems are seen through their final words: a user sends a request, the model generates a reply, and the reply is judged by clarity, usefulness, tone, and accuracy.

Garmon asks a deeper question:

> what happens before the answer becomes words?

The central idea is that speech should not be the first body of the system. Before speech, there should be a path: internal state, body contour, meaning, choice boundary, trace, and only then expression.

A compact working chain is:

```text
body -> body contour -> meaning -> choice boundary -> trace -> speech
```

In Garmon, the LLM is not treated as the whole system. It is treated as the speech layer: the organ that turns an already shaped direction into human-readable language.

This repository contains public research materials, public-safe summaries, and reviewable boundaries for that work.

## The Core Question

A strong answer is not enough by itself.

A text can be precise, warm, intelligent, and convincing while still leaving the main question open:

> where did this answer come from?

If the answer begins only at the level of generated text, then we see the surface. Garmon studies the possibility of a deeper order: an input first meets an internal state, changes a body contour, receives meaning, approaches a protected choice boundary, leaves a trace, and only then becomes speech.

This is the central shift:

```text
request -> analysis -> plan -> answer
```

is not the only possible public shape.

Garmon studies another one:

```text
internal state -> body contour -> meaning -> choice boundary -> trace -> LLM speech
```

The goal is not to make language sound more dramatic.

The goal is to make the origin of the answer more visible, more bounded, and less dependent on speech pretending to be the whole system.

## What Garmon Holds

Garmon holds a layered order of answer formation.

Each layer has a role. Each layer also has a boundary.

- Internal state gives the first background.
- Body contour shows how an event changes that background.
- Meaning reads the event relative to the current line.
- Choice boundary protects the place where one path may become the result.
- Trace keeps the past from disappearing without letting it rule the present.
- Speech expresses the path without becoming its source.
- Evidence supports boundaries without becoming authority.

The short law is:

```text
participation is not power
```

Body participates, but does not choose.

Meaning participates, but does not command.

Memory participates, but does not become a hidden motor.

Observation participates, but does not become the owner of truth.

Speech participates, but does not become the source of choice.

Evidence participates, but does not open the next layer by itself.

This separation is not a disclaimer. It is part of the architecture.

## Plain-Language Version

Garmon studies what happens before an LLM writes the final answer.

In a normal chatbot, the request often goes straight into text generation. The final reply may be useful, but the step where the event receives internal meaning is usually hidden or collapsed into the text itself.

Garmon studies a different order.

First, the input is treated as an event. That event meets an internal state. The state gives the event a contour: tension, recovery, caution, openness, fatigue, pressure, or another kind of internal weight.

From that contour, a meaning direction can appear.

Only after that should speech begin.

In simple terms:

> Garmon is not trying to make text look alive.  
> Garmon studies whether an answer can grow from an internal path before it becomes text.

## Body-Born Meaning

This repository uses the term **body-born meaning** for a public, high-level idea:

> an event receives meaning through an internal-state layer before final LLM speech.

This is not a biological claim.

It does not mean that Garmon copies the human body.

It means that the system is not treated as an empty surface waiting for text. The event meets an internal background first, and that meeting can change the direction of the answer.

Body-born meaning is important because it gives the answer weight before wording.

Without this layer, the system may still speak well, but the origin of the answer remains unclear.

## Speech Has a Place

Speech is the most visible layer of the system.

It is also easy to overestimate.

A good sentence can look like understanding. A confident explanation can look like choice. A polished answer can look like inner continuity.

Garmon keeps speech in its proper place.

Speech is an organ of expression. It makes the internal path visible to a human reader. It can clarify, soften, strengthen, explain, and translate a selected direction into language.

But speech must not create the choice after the fact.

Speech expresses the path. It does not replace the path.

```text
speech expresses
speech does not originate
speech makes visible
speech does not own
```

## Memory as Trace

Garmon treats memory as trace, not as a hidden motor.

The past matters because a system should not begin every event from an empty screen. A previous event can leave caution, fatigue, trust, recovery, resistance, or a changed sensitivity.

But the past must not choose instead of the present.

Memory has a lawful role:

```text
past event -> trace -> changed sensitivity -> present meaning
```

It does not become:

```text
past event -> hidden command -> current result
```

This distinction is central to Garmon.

A system without trace is too flat.

A system where memory becomes a second chooser is no longer one organism.

## One Truth Path

Garmon separates three things:

```text
decision
trace of decision
story about decision
```

A report is not the decision.

A log is not the decision.

A memory trace is not the decision.

A spoken explanation is not the decision.

This matters because a complex system can easily create several small centers of authority: one layer says what happened, another stores it, another explains it, another uses it later, and speech makes it sound coherent.

Garmon’s architecture protects a single truth path.

The point is not bureaucracy. The point is integrity.

An answer should not be assembled from competing centers that all pretend to be the source.

## Why This Matters

Many questions in AI behavior are hidden when we inspect only the final answer.

A final answer can be useful and still tell us very little about its origin.

Garmon makes a different question visible:

> was there a path before speech?

This matters for:

- safer response shaping;
- long-running assistants;
- believable digital characters;
- game and NPC systems;
- human-facing AI companions;
- robotics interfaces;
- AI safety review;
- studying the difference between text behavior and behavior before text.

The immediate public value of Garmon is not louder language.

The value is a clearer review surface for the pre-speech path.

## One-Minute Example

A tense message arrives.

A plain text-task shape might look like this:

- the system treats the message as a request;
- the model writes a polite reply;
- the reply may be useful;
- the internal meaning of the event remains hidden.

A Garmon-shaped public example looks different:

- the message first enters as an event;
- the event meets an internal state;
- the state gives the event a body contour;
- the contour creates a meaning direction;
- the choice boundary protects what may become the result;
- the LLM speech layer writes from that direction.

The point is not “smarter text.”

The point is answer origin before speech.

## What Is Currently Supported

Garmon is shared here as a public research preview.

The current public-safe evidence supports a bounded lower path before speech.

Static and shadow checks show that several body-contour candidates can remain distinguishable before LLM speech, including quiet baseline, fatigue, recovery, stress, mixed stress/recovery, and day/night rhythm.

This matters because these contours do not collapse into one generic label.

The public evidence also supports an important boundary:

```text
body contour -> meaning candidate -> passive context -> choice visibility -> choice ownership -> behavior boundary -> pre-selected result boundary candidate
```

The current achievement is not a finished live system.

The achievement is that part of the pre-speech path can be described, inspected, and bounded without giving the wrong layer the wrong authority.

The body contour does not become a command.

The meaning candidate does not become a selected result.

The trace does not become a chooser.

The report does not become truth.

The speech layer does not become will.

## Boundaries of Claim

Garmon does not need to begin with negation.

It begins with a positive claim:

> an artificial answer can be studied as something that has an origin before speech.

To keep that claim honest, this repository separates three zones.

### Supported

The public materials support the layered discipline of the project:

- answer is not reduced to the final phrase;
- internal state can shape the reading of an event;
- body-contour candidates can remain distinguishable before speech;
- memory is treated as trace, not as motor;
- speech is treated as expression, not as the source;
- evidence supports boundaries, not authority.

### Theory

The broader theory says that artificial behavior should be studied through a path:

```text
state -> body contour -> meaning -> choice boundary -> trace -> speech
```

This theory gives language for asking where an answer came from, not only whether it sounded good.

### Open Hypotheses

Some questions remain future work:

- how body-born meaning may participate in choice without becoming a hidden command;
- how memory can become deeper experience while remaining trace, not motor;
- how expected state should be built across time rather than from one snapshot;
- how body tendency should be studied as trajectory, not as a single frame;
- how speech can become a more precise organ of expression without becoming the source of choice.

These open questions are not weaknesses.

They are the working frontier of the project.

## Future Direction

Garmon should grow cleaner by layers, not wider by noise.

Future work focuses on:

- internal time and continuity;
- memory as trace without memory becoming a second chooser;
- a cleaner choice boundary;
- speech as a more precise organ of expression;
- external review by people who can challenge the theory;
- open language that is strong without becoming sensational.

The rule for future growth is simple:

```text
growth should strengthen the organism
growth should not create a second owner
```

If a new layer makes meaning more precise without stealing choice, it is useful.

If memory makes the path longer without commanding the present, it is useful.

If evidence reveals a boundary without becoming permission, it is useful.

If speech expresses the internal path without claiming to be its source, it is useful.
