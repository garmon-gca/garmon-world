# Public Engineering Contracts

This page defines the public engineering boundaries for Garmon.

## Core Public Model

Public shape:

```text
event → public event reading → response direction → LLM speech → final answer
```

Broader research frame:

```text
state → body contour → meaning → choice boundary → trace → speech
```

This separates layers: event, internal state, body contour, meaning, choice boundary, trace, speech.

## Main Engineering Law

```text
participation is not power
```

Each layer participates without owning the final answer.

- state shapes reading
- body contour gives weight
- meaning appears before wording
- choice boundary protects selection
- trace preserves influence without control
- speech expresses the path

## Current Public Achievement

Body Shadow delivers distinguishable body contours before speech:

- quiet baseline
- fatigue
- recovery
- stress
- mixed stress/recovery
- day/night rhythm

Contours stay separate. This opens a pre-speech review surface.

## Static Choice Boundary Chain

```text
body contour → meaning candidate → passive context → choice visibility → choice ownership → behavior boundary
```

This chain marks the path. It names the boundary before selected result.

7. Response direction guides speech.
8. Response direction stays before selected result.
9. Trace preserves past influence as soft pressure.

11. Body contour stays distinct from command, memory, behavior, truth, speech.
12. Body Shadow creates review surface.
13. Static/shadow evidence supports distinction.
14. Negative controls catch authority leaks.
15. Visibility does not equal authority.
16. Influence does not equal choice.
17. Passive context supports review without control.
18. Public artifacts stay small, bounded, reviewable.
19. Public materials separate evidence, theory, open hypothesis.
20. Public explanations show principle without exposing mechanism.
