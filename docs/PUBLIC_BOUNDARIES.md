# Public Boundaries

This repository is public-facing. It should stay readable, safe, and achievement-aware.

The current public layer should show what has become inspectable without publishing the private mechanism.

The goal is to keep Garmon understandable for outside readers while preserving clear boundaries around internal state, meaning, memory, choice, behavior, truth, and speech.

## Safe to Discuss

It is safe to discuss:

- Garmon as an early R&D prototype.
- Garmon as a public research preview, not an interactive demo.
- Body-born meaning at a high level.
- The path before final LLM speech.
- Internal state as a pre-speech layer.
- Event reading as a public review surface, not as a command or private mechanism.
- Body Shadow as a public-safe static check surface.
- Body-contour distinction before LLM speech.
- Quiet baseline, fatigue, recovery, stress, mixed stress/recovery, and day/night rhythm as public-safe contour examples.
- Negative controls as a way to show that forbidden authority leaks are caught.
- The LLM as a speech layer.
- The difference between an event-reading step, response direction, and final wording.
- Memory or residue as future-gated evidence, not as a hidden motor.
- Public-safe capsule as an outward container, not as the organism.
- Controlled implementation gate as a public boundary principle.
- Public artifact and demo planning.
- The closed static choice chain and the passive chain reaching a pre-selected result boundary candidate in synthetic evidence, while emphasizing that the door remains closed and no actual selected result is claimed.
- The broader working direction: internal state, meaning, choice boundaries, time/residue, and speech should remain separate.

## Boundary Rules

Public materials should preserve these boundaries:

- internal state is not command authority;
- body contour is not command authority;
- event reading is not a command;
- meaning candidate is not behavior;
- passive context is not chooser input;
- choice visibility is not live choice;
- choice ownership is not transferred to context;
- memory or residue is not a hidden motor;
- speech is not the source of choice;
- evidence is not decision truth;
- visibility is not authority;
- static/shadow evidence is not runtime proof;
- a pre-selected result boundary candidate is not an actual selected result;
- public summaries are not the private mechanism;
- a public-safe capsule is an outward container, not the organism, not memory, not truth, and not the chooser.

## Tone Rule

Use achievement-first careful language.

Good:

> Body Shadow shows distinguishable body-contour candidates before speech.

> The boundary is preserved: contours do not become commands.

> Garmon studies the path before final speech without presenting static/shadow evidence as live behavior.

> The pre-selected result boundary is marked, but it is not opened.

Avoid:

> finished system

> solved autonomy

> final evidence

> urgent major discovery

> live behavior from static checks

> conscious system

> living AI

> memory-driven behavior

> speech creates choice

> public proof of life

## First Public Goal

The first goal is to show a small inspectable achievement without overstating it.

The current achievement is Body Shadow: body-contour candidates can be inspected before speech while remaining separate from command, memory, behavior, truth, speech authority, selected-result authority, runtime proof, and implementation permission.

The static choice chain closure and the synthetic pre-selected result boundary candidate extend this achievement without opening the door to actual selection or live behavior.

The goal is better technical review, not hype.

## One-Line Summary

The public surface may show that Garmon has a bounded, inspectable pre-speech review layer, but it must not imply live behavior, actual selected results, memory authority, speech authority, consciousness, life, or a finished autonomous system.
