# Public Boundaries

This repository is public-facing.

It should stay readable, safe, and achievement-aware.

The public layer should show what has become inspectable without publishing the private mechanism.

The goal is to keep Garmon understandable for outside readers while preserving clear boundaries around internal state, body contour, meaning, memory, choice, behavior, truth, evidence, and speech.

## Core Public Principle

Garmon studies answer origin before speech.

A compact public frame is:

```text
state -> body contour -> meaning -> choice boundary -> trace -> speech
```

This frame can be discussed publicly.

The private working contour is not published here.

Public materials should explain the principle, the current public-safe achievement, and the boundaries that keep the claim honest.
