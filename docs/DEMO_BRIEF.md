# Demo Brief

This document outlines the direction for future public Garmon demo artifacts.

The next public artifact focuses on a small, bounded, reviewable piece that shows what is already visible in the public-safe layer.

## Core Goal

The first public artifact answers one narrow question:

> Can body-contour candidates remain distinguishable before final LLM speech?

A later useful question:

> Can an event receive a visible public reading before final speech?

The broader public Garmon direction:

```text
state → body contour → meaning → choice boundary → trace → speech
```

## Current Public Achievement

Body Shadow provides the clearest public-safe achievement.

Static and shadow checks show several body-contour candidates remain distinguishable before speech:

- quiet baseline
- fatigue
- recovery
- stress
- mixed stress/recovery
- night/downshift
- day/wake support

These contours stay separate. This creates a meaningful pre-speech review surface.

## Best First Artifact

A body-contour distinction artifact.

It stays small, reviewable, bounded, public-safe, and understandable in minutes.

The artifact shows:

- a small set of contours
- one public-safe reading for each
- visible distinction between contours
- clear limits
- clear boundary

Achievement comes first. Boundary stays visible right after.

## Example Artifact Shape

```text
quiet baseline → stable low-pressure contour
fatigue → low-energy caution contour
recovery → downshift / return contour
stress → tension contour
mixed stress/recovery → recovery under remaining pressure
night/downshift → body-time downshift contour
day/wake support → body-time wake-support contour
```

Review question: Do these contours remain distinguishable before speech?

The value lies in inspectable distinction.

## Event-Reading Direction

Later artifacts explore meaning before speech more directly.

Example structure:

- one visible event
- one public-safe event reading
- two close response directions
- one direction chosen
- final answer following that direction
- comparison note
- limits section

Comparison:

```text
direct text response
vs
event → reading → response direction → speech
```

The event-reading layer stays reviewable. It does not become command authority.

## What The Artifact Shows

The artifact helps reviewers see:

- which body-contour candidates are compared
- what public-safe reading each receives
- whether contours remain distinct
- what the result suggests
- what the result does not extend to

For event-reading:

- the event
- the reading before speech
- the response direction used
- what changed between directions
- how speech followed the direction

The artifact works without private mechanism access.
