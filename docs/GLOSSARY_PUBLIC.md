# Public Glossary

This glossary explains public terms used in Garmon World. It avoids private implementation details.

## Bodily background

The public term for the inner background from which Garmon reads an event.

This does not mean a published biological simulation or a human-like body.

## Body-born meaning

The meaning an event receives before final speech.

In Garmon World, this is the central public idea: the request is not only text to process, but an event that can be read before the LLM writes final words.

This is a public research term for a high-level event-reading step.

It is not a claim of biological life, subjective experience, consciousness, or a finished autonomous system.

## Body Shadow

A public term for static checks that examine whether body-contour candidates remain distinct before speech.

It is a review surface, not a claim of live interaction, live behavior, subjective experience, or runtime proof.

## Body contour

A public-safe contour such as quiet baseline, fatigue, recovery, stress, mixed stress/recovery, or day/night rhythm.

A body contour is not:

- a command;
- memory;
- behavior;
- truth;
- final speech;
- chooser authority;
- implementation permission.

## Pre-speech layer

The public term for the layer before the LLM writes final text.

In Garmon World, the pre-speech layer may include public-safe descriptions of internal state, event reading, meaning candidates, response direction, and boundaries.

It is not a public claim of live choice or private runtime behavior.

## Event reading

The public term for how an event is interpreted before final speech.

An event reading is not a command, not final behavior, and not the private mechanism.

## Meaning candidate

A public-safe candidate for what an event may mean before final speech.

A meaning candidate is not decision truth and not behavior.

## Response direction

The public name for the direction that final speech should follow.

It is not the same as final speech.

It is also not a public claim of live choice or actual selected result.

## LLM speech layer

The part of the system where an LLM writes the final text shown to a user.

The LLM speech layer is not presented here as the whole system or as the source of choice.

## Line of behavior

A public phrase for a continuing direction of behavior over time, rather than a single isolated answer.

In this repository, it is used cautiously. It does not mean proven consciousness, personhood, biological life, or mature autonomy.

## Memory as residue

A cautious public phrase for the idea that memory-like material should be treated as a trace or residue, not as a hidden motor.

Residue may be discussed as future-gated evidence, but this repository does not claim active body memory, memory influence on choice, or lived biography.

## Residue

A possible future trace left by an event after recovery and review.

A residue-like claim would require separate evidence, provenance, and boundary review.

A diagnostic tail, warning, static check, or public artifact is not automatically residue.

## Static check

A non-runtime check that can show distinction without opening live interaction.

A static check is not runtime proof.

## Shadow check

A bounded review check that inspects a public-safe shadow surface without publishing or activating the private mechanism.

A shadow check is not live behavior.

## Static/shadow evidence

Public-safe evidence from static or shadow checks.

It may support reviewability and boundary clarity, but it does not prove consciousness, biological life, mature autonomy, live behavior, or runtime execution.

## Negative control

A deliberately invalid case used to check that forbidden authority leaks are caught.

A useful check should pass valid cases and fail when a forbidden authority leak is introduced.

## Passive context

A public term for context that can be visible for review without becoming chooser input or behavior authority.

Passive context is not influence by itself.

## Choice visibility

A public term for making a choice-related boundary visible for review.

Choice visibility is not live choice.

## Choice ownership boundary

A public term for the boundary that prevents ownership of choice from being transferred to context, speech, reports, or memory.

It is not a claim that public live choice has been opened.

## Behavior boundary

A public term for the boundary before behavior can be claimed.

Context alone does not create behavior.

A behavior boundary is not public live behavior.

## Static choice chain

The sequence of pre-speech layers from body contour through meaning candidate, passive context, choice visibility, choice ownership, and behavior boundary.

In public terms, the static choice chain is a layer of static/shadow evidence: it verifies that body-contour candidates can move along this path without becoming commands, memory, behavior, truth, speech authority, selected-result authority, runtime proof, or implementation permission.

## Pre-selected result boundary

A public term for the synthetic marker just before an actual selected result.

In recent evidence, the passive chain has been extended up to a pre-selected result boundary candidate.

This indicates that the door next to selection has been named, but it remains closed.

No actual selected result, choice, behavior, application authority, memory influence, or live runtime has been opened.

This boundary is evidence only and does not grant implementation permission.

## Public-safe capsule

A safe outward container for representation after internal selection.

It is not:

- the organism;
- memory;
- truth;
- the chooser;
- life-status evidence;
- consciousness evidence.

## Controlled implementation gate

A rule that review evidence is not implementation permission.

Implementation requires explicit human approval and a narrow scoped gate.

## Visibility

The fact that something can be seen, inspected, or summarized.

Visibility is not authority.

## Influence

A possible effect on interpretation or context.

Influence is not automatically authority or choice.

## Authority

The right or power of a layer to affect a decision, behavior, memory, or implementation.

Authority must not be implied from visibility alone.

## Decision truth

The source that would count as the actual decision truth inside the private system.

Public reports, summaries, artifacts, and evidence notes are not decision truth.

## Public artifact

A small reviewable output that can be inspected without private internals.

The first useful artifact does not have to be an interactive demo.

A public artifact should remain small, bounded, and public-safe.

## Public-safe

Safe for public description without exposing private files, private mechanisms, raw logs, local paths, runtime internals, protected hashes, hidden channels, or exact implementation details.

Public-safe does not mean complete.

## One-line summary

Garmon World uses these terms to describe a bounded public review surface: body contours and event readings can be inspected before final LLM speech without becoming command, memory, behavior, truth, speech authority, selected-result authority, runtime proof, or implementation permission.
