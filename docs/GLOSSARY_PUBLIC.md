# Public Glossary

This glossary defines key public terms in Garmon World.

## Answer origin before speech

Central public idea.

An artificial answer has a path before it becomes final words.

Frame:

```text
state → body contour → meaning → choice boundary → trace → speech
```

## Internal state

Inner background that shapes event reading before speech.

## Body contour

Shape of internal state around an event.

Examples:

- quiet baseline
- fatigue
- recovery
- stress
- mixed stress/recovery
- day/night rhythm

Contours stay distinguishable before speech.

## Body-born meaning

Meaning an event receives through internal state before LLM speech.

## Body Shadow

Static and shadow checks that show body-contour distinction before speech.

Creates public review surface.

## Pre-speech layer

Layer before final LLM text.

Includes state, contour, event reading, meaning candidate, response direction, boundaries.

## Event

Incoming situation or request read through internal state.

## Event reading

Interpretation of event before speech.

Shapes response direction.

## Meaning candidate

Possible meaning before final selection.

## Response direction

Direction final speech follows.

Example: lower intensity, keep contact without pressure.

## LLM speech layer

Layer that produces final readable text.

## Speech

Visible language output that expresses the path.

## Trace

Mark left by past event.

Soft pressure: changed sensitivity, caution, recovery.

## Memory as trace

Past influence as soft pressure, not motor.

## Residue

Future trace after recovery and review.

Future-gated.

## Line of behavior

Continuing direction over time.

## Static check

Non-runtime check for distinction and boundary preservation.

## Shadow check

Bounded review on public-safe surface.

## Static/shadow evidence

Evidence from checks supporting distinction.

## Negative control

Invalid case that tests boundary enforcement.

## Passive context

Context visible for review without control.

## Visibility

Ability to inspect.

Separate from authority.

## Influence

Effect on interpretation.

Separate from choice.

## Authority

Power to decide or control.

Participation separate from power.

## Choice boundary

Protected place where path becomes result.

## Choice visibility

Making boundary visible for review.

## Behavior boundary

Boundary before behavior claim.

## Static choice boundary chain

```text
body contour → meaning candidate → passive context → choice visibility → choice ownership → behavior boundary
```

## Pre-selected result boundary candidate

Marker before actual selected result.

Names the boundary.

## Public artifact

Small reviewable output.

Small, bounded, public-safe.

## One-line summary

Garmon World terms describe distinguishable layers before speech: body contours, event readings, response directions. Review surface stays clear and bounded.
