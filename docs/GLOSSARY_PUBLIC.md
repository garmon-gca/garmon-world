# Public Glossary

This glossary explains public terms used in Garmon World. It avoids private implementation details.

## Bodily background

The public term for the inner background from which Garmon reads an event. This does not mean a published biological simulation or a human-like body.

## Body-born meaning

The meaning an event receives before final speech. In Garmon World, this is the central public idea: the request is not only text to process, but an event that can be read before the LLM writes final words.

This is a public research term for a high-level event-reading step. It is not a claim of biological life, subjective experience, or a finished autonomous system.

## Body Shadow

A public term for static checks that examine whether body-contour candidates remain distinct before speech. It is a review surface, not a claim of live interaction.

## Body contour

A public-safe contour such as quiet baseline, fatigue, recovery, stress, mixed stress/recovery, or day/night rhythm. It is not a command, not memory, not behavior, not truth, and not final speech.

## Static check

A non-runtime check that can show distinction without opening live interaction.

## Negative control

A deliberately invalid case used to check that forbidden authority leaks are caught.

## Passive context

A public term for context that can be visible for review without becoming chooser input or behavior authority. Passive context is not influence by itself.

## Public-safe capsule

A safe outward container for representation after internal selection. It is not the organism, not memory, not truth, not the chooser, and not life-status evidence.

## Controlled implementation gate

A rule that review evidence is not implementation permission. Implementation requires explicit human approval and a narrow scoped gate.

## Response direction

The public name for the direction that final speech should follow. It is not the same as final speech.

## LLM speech layer

The part of the system where an LLM writes the final text shown to a user. The LLM speech layer is not presented here as the source of choice.

## Static choice chain

The sequence of pre-speech layers from body contour through meaning candidate, passive context, choice visibility, choice ownership, and behavior boundary.

In public terms, the static choice chain is a layer of static/shadow evidence: it verifies that body-contour candidates can move along this path without becoming commands, memory, behavior, truth, or implementation permission.

## Pre-selected result boundary

A public term for the synthetic marker just before an actual selected result. In recent evidence, the passive chain has been extended up to a pre-selected result boundary candidate. This indicates that the door next to selection has been named, but it remains closed.

No actual selected result, choice, behavior, application authority, memory influence, or live runtime has been opened. This boundary is evidence only and does not grant implementation permission.

## Public artifact

A small reviewable output that can be inspected without private internals. The first useful artifact does not have to be an interactive demo.
