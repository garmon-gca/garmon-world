# License Status

No open-source license has been selected yet.

This means the public materials in this repository are currently shared for reading, orientation, and review, but not as a general reuse grant.

## Why No License Yet

Garmon World is still in an early public research-preview stage.

The repository includes public-safe research notes, including Body Shadow, body-contour distinction before speech, static/shadow boundary summaries, and public-facing documentation.

The public scope is still being shaped.

The project needs a clearer split between:

- public notes;
- review materials;
- future public artifacts;
- possible future demo code;
- brand assets;
- public examples;
- public diagrams;
- private workbench material that must not be published.

Choosing a license too early could create confusion about what may be reused, copied, modified, redistributed, or treated as software.

## Current Practical Meaning

For now:

- reading is welcome;
- calm technical feedback is welcome;
- linking to the repository is welcome;
- quoting short parts with attribution is reasonable;
- discussing the public ideas is welcome;
- using the repository as a reference for review is welcome;
- copying the repository as a reusable software package is not the current intent;
- reusing the name, brand assets, diagrams, project framing, or public artifacts as if they were open assets is not granted here.

## What Is Not Granted

Until a license is selected, this repository does not grant permission to:

- reuse Garmon branding as your own;
- redistribute the repository as a software package;
- copy public artifacts into another project as reusable assets;
- treat future demo code as open-source by default;
- treat public diagrams or examples as open reusable templates;
- imply endorsement, partnership, or shared ownership;
- publish private or reconstructed implementation details.

## Future License Decision

A license may be added later when the public scope is clearer.

Possible future split:

- documentation license for public notes;
- separate software license for demo code if code is added;
- separate handling for brand assets;
- separate handling for diagrams;
- separate handling for public artifacts and review examples;
- separate handling for any future dataset-like material, if such material is ever added.

Until then, please treat this repository as public research notes and review materials, not as an open-source software release.

## One-Line Summary

Garmon World is public for reading, orientation, linking, and calm technical review, but no general reuse, software, brand, artifact, or redistribution license has been granted yet.
