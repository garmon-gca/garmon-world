# One-Minute Example

Это объясняющий пример идеи Garmon:

> ответ может иметь происхождение до речи.

## Проблема

LLM пишет полезный текст.

Garmon изучает pre-speech layer.

Компактная рамка:

```text
state → body contour → meaning → choice boundary → trace → speech
```

## Пример 1: Чтение события

Событие: напряжённое сообщение.

Garmon-форма:

```text
lower intensity
keep contact without pressure
avoid escalation
```

Response direction: calm boundary reply.

Публичная цепочка:

```text
event → public event reading → response direction → LLM speech
```

## Пример 2: Body-Contour Distinction

```text
quiet baseline → stable low-pressure
fatigue → low-energy warning
recovery → return contour
stress → tension
mixed → tension plus return
night → downshift
```

Body Shadow делает distinction reviewable.

## Пример 3: Граница

```text
body contour → meaning candidate → passive context
```

```text
participation is not power
```

## Пример 4: Речь

```text
response direction → LLM speech → final answer
```

## One-Line Summary

event → body-contour-shaped reading → response direction → LLM speech
