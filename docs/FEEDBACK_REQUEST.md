# Feedback Request

This repository is a public research preview, not an interactive demo.

Garmon is an early research prototype studying answer origin before speech.

The current public-safe achievement is **Body Shadow**: static/shadow checks show that several body-contour candidates remain distinguishable before LLM speech.

Public-safe contour examples include:

- quiet baseline;
- fatigue;
- recovery;
- stress;
- mixed stress/recovery;
- day/night rhythm.

These contours do not collapse into one generic label.

The useful point is not stronger wording.

The useful point is a more inspectable pre-speech layer.

## What Garmon Studies

Garmon studies the path before final LLM wording.

A compact public frame is:

```text
state -> body contour -> meaning -> choice boundary -> trace -> speech
```

This does not claim live choice, live behavior, consciousness, life, or mature autonomy.

It is a way to keep the path before final speech reviewable without collapsing state, meaning, memory, choice, behavior, truth, evidence, and speech into one text-generation step.

The LLM remains important.

It is the speech layer: it writes the final words, but it is not presented as the whole system or as the source of choice.

## Current Public-Safe Result

The current public-safe result is narrow but useful:

> body-contour candidates can remain distinguishable before final speech.

This gives Garmon a reviewable surface before final wording.

The boundary remains clear:

- body contour is not command;
- body contour is not memory;
- body contour is not behavior;
- body contour is not truth;
- body contour is not final speech;
- body contour is not selected-result authority;
- body contour is not implementation permission.

The central public law is:

```text
participation is not power
```

A contour may participate in the path without owning the answer.
