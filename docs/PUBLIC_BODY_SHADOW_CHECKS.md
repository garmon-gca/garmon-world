# Public Body Shadow Checks

Status: public-safe summary of static/shadow Body Shadow checks.

This page does not publish private implementation details.

## Core Result

Body Shadow gives Garmon its first public-safe body-contour distinction surface.

The current public result is simple:

> several body-contour candidates can remain distinguishable before final LLM speech.

This matters because the final answer alone may hide whether different internal-state contours were preserved or collapsed into the same generic response shape.

The achievement is not stronger wording.

The achievement is inspectability before speech.

## What Has Been Shown

Static/shadow checks distinguish several body-contour candidates before LLM speech.

Public-safe checked contours include:

- quiet baseline;
- neutral body state;
- fatigue;
- recovery;
- stress;
- mixed stress/recovery;
- night/downshift;
- day/wake support;
- calm-body request context;
- tense-body request context.

The important result is that these contours do not collapse into one generic label.

They remain distinguishable before final text is written.

## Why This Matters

Garmon studies answer origin before speech.

A compact public frame is:

```text
state -> body contour -> meaning -> choice boundary -> trace -> speech
```

Body Shadow supports the lower part of this direction by making body-contour differences reviewable before final wording.

A smaller public-safe chain for this page is:

```text
body contour -> meaning candidate -> passive context -> speech boundary
```

This does not claim live choice.

It does not claim live behavior.

It does not claim consciousness, biological life, subjectivity, or mature autonomy.

It shows that the pre-speech layer can preserve meaningful contour differences before final LLM speech.

## Negative Controls

The checks include negative controls.

A negative control is a deliberately invalid case that should fail.

This matters because a useful public-safe check must not only recognize valid contour summaries. It must also reject cases where a contour tries to take a role it should not have.

In public terms, a useful check should fail when a body contour is framed as:

- command;
- memory;
- behavior;
- truth;
- final speech;
- selected-result authority;
- runtime proof;
- implementation permission.

This helps preserve the central Garmon law:

```text
participation is not power
```

A contour may participate in the pre-speech path.

It must not become the owner of the answer.

## Relation to Meaning

Body Shadow does not claim that a body contour is already final meaning.

A safer public chain is:

```text
body contour -> meaning candidate
```

The word “candidate” matters.

It means the contour may support a direction of meaning, but it does not become behavior, selected result, command, or final truth.

A body contour can give weight to an event.

It cannot choose the result by itself.

## Relation to Speech

Body Shadow happens before final speech.

The LLM remains the speech layer.

Speech may later express a response direction in human language, but Body Shadow does not make speech the source of choice.

Correct public framing:

```text
body contour before speech
speech after response direction
```

Incorrect framing:

```text
speech creates the contour
speech creates the choice
```

## Relation to Memory

Body Shadow is not memory.

It does not turn body-contour candidates into lived residue, biography, or active memory authority.

A body-contour review surface may help inspect the pre-speech layer.

But inspectability is not memory.

Public-safe distinction:

```text
visible contour is not lived memory
review surface is not biography
```
